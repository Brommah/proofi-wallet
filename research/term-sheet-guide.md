### 2.1 Valuation (Pre-Money vs Post-Money)

#### How It Works

```
Post-Money Valuation = Pre-Money Valuation + Investment Amount
Investor Ownership = Investment Amount Ã· Post-Money Valuation
Founder Dilution = Investment Amount Ã· Post-Money Valuation
```

**Example:**
```
Pre-money: $20M
Investment: $5M
Post-money: $25M
Investor ownership: $5M Ã· $25M = 20%
```

#### The Option Pool Shuffle

**Critical concept:** Option pool increases from pre-money dilute founders, not investors.

```
Without pool expansion:
Pre-money: $20M (founders own 100%)
Investment: $5M
Post-money: $25M
Founder ownership: $20M Ã· $25M = 80%

With 15% pool from pre-money:
Pre-money: $20M (85% founders + 15% pool)
Investment: $5M  
Post-money: $25M
Founder ownership: $17M Ã· $25M = 68%
Effective valuation: $17M
```

**Non-Participating Preferred (Standard):**
```
Investor chooses EITHER:
(a) Get 1x investment back, OR
(b) Convert to common and share pro-rata

Exit at $10M (1x): Investor takes $5M preference

(vs $10M with non-participating)

```
Investor gets preference + participation UP TO cap (e.g., 3x)

### 2.3 Anti-Dilution Protection

#### Broad-Based Weighted Average (Standard)

Protects investors if company raises at lower price ("down round").

```
New Price = Old Price Ã— (A + B) / (A + C)

A = Pre-issuance fully diluted shares
B = Shares that WOULD be issued at original price
C = Shares ACTUALLY issued at new (lower) price

Example:
Series A at $1/share, 10M shares outstanding
Down round at $0.50/share, raising $2M (4M new shares)

A = 10M
B = $2M Ã· $1 = 2M
C = $2M Ã· $0.50 = 4M

New Price = $1 Ã— (10M + 2M) / (10M + 4M) = $0.857
Series A conversion price adjusts from $1 to $0.857
Series A gets ~17% more shares upon conversion
```

#### Full Ratchet (Investor-Friendly, Avoid)

```
Any down round = Series A price resets to new price entirely

Same example:
Series A price resets from $1 to $0.50
Series A gets 2x the shares upon conversion
Massively dilutive to founders
```

#### Pay-to-Play

Requires investors to participate in down rounds to keep anti-dilution protection.

```
RECOMMENDED ADDITION:

Pay-to-Play: Investors must purchase their pro-rata share of any
down-round financing to maintain anti-dilution protection. 
Non-participating investors convert to Common or shadow preferred.
```

## 3. Negotiation Tactics

### 3.1 What to Push Back On

#### High Priority (Always Negotiate)

| Term | Why | Target |
|------|-----|--------|
| **Participating preferred** | Massive downside in exits | Non-participating 1x |
| **Full ratchet anti-dilution** | Destroys cap table in down round | Broad-based weighted average |
| **Super pro-rata rights** | Blocks future investors | Standard pro-rata only |
| **Excessive option pool** | Stealth valuation reduction | Pool based on hiring plan |
| **Founder vesting restart** | Unfair to existing work | Credit for time served |
| **Redemption rights** | Forced exit pressure | Remove or 7+ year term |

#### Medium Priority (Negotiate If Possible)

| Term | Why | Target |
|------|-----|--------|
| **Board seat for small check** | Over-representation | Board seat requires lead/major check |
| **Broad protective provisions** | Operational friction | Limited list, higher thresholds |
| **Information rights for all** | Administrative burden | Major investors only (>$250K) |
| **Founder non-competes** | Limits future options | 12 months max, narrow scope |
| **Milestone-based tranches** | Execution risk on company | Single close, or founder-friendly milestones |

### 3.2 What to Accept

#### Generally Acceptable

| Term | Why Acceptable |
|------|----------------|
| **1x non-participating liquidation preference** | Industry standard, fair |
| **Broad-based weighted average anti-dilution** | Standard protection |
| **Standard pro-rata rights** | Reasonable for committed investors |
| **4-year vesting with 1-year cliff** | Industry norm |
| **Board seat for lead investor** | Reasonable governance |
| **Standard information rights** | Transparency is healthy |
| **ROFR and co-sale** | Protects all parties |
| **D&O insurance requirement** | Protects everyone |

#### Accept with Conditions

| Term | Condition |
|------|-----------|
| **Drag-along** | Require minimum value threshold (2x preference) |
| **No-shop** | 30 days max, narrow scope |
| **Founder reps** | Limit to actual knowledge, cap liability |
| **Pay-to-play** | Actually founder-friendly if structured right |

## Appendix B: Key Formulas

### Valuation Math

```
Post-Money = Pre-Money + Investment
Ownership % = Investment Ã· Post-Money
Effective Pre-Money = Pre-Money - Pool Increase Value
Price/Share = Pre-Money Ã· Fully Diluted Shares (pre-close)
```

### Dilution

```
New Ownership = Old Ownership Ã— (Old Post / New Post)
Dilution = 1 - (New Ownership / Old Ownership)
```

### Anti-Dilution (Broad-Based Weighted Average)

```
Adjusted Price = Original Price Ã— (A + B) / (A + C)

A = Fully diluted shares pre-issuance
B = Shares at original price from new $: New $ Ã· Original Price
C = Shares actually issued: New $ Ã· New Lower Price
```

### Liquidation Waterfall

```
Non-Participating:
Preferred Payment = MIN(Preference, Exit Value)
Common Payment = MAX(0, Exit Value - Preference)
OR if converted: Everyone shares pro-rata

Participating:
Preferred Payment = Preference + (Exit - Preference) Ã— Preferred%
Common Payment = (Exit - Preference) Ã— Common%
```
