## About Cere Network

Cere is a **decentralized data infrastructure** platform designed for the AI data era:
- Edge computing for AI data processing
- Decentralized Data Cloud (DDC)
- Web3 x AI convergence (AI Agents, Edge Clusters)
- Community-governed via Cere DAO
- $CERE token on Polkadot ecosystem
- Use cases: Gaming (CerePlay), Music (DaVinci), AI agent orchestration

**Key Value Props for Investors:**
- Solving the AI data bottleneck (training data scarcity)
- Infrastructure for decentralized AI/ML pipelines
- Edge computing reduces central storage costs
- Real-world adoption (gaming, media, AI)

### Tier 1: Warm Intros (Perfect Fits)
1. **Leverage existing relationships** - Check if advisors/team have connections
2. **Portfolio company intros** - Ask comparable portfolio companies (Grass, IO.net, Celestia founders)
3. **Conference networking** - ETHDenver, Token2049, Consensus

### Tier 2: Cold Outreach Best Practices
1. **Twitter DM** - Most partners are responsive on Twitter
2. **Email** - Use investor submission forms where available
3. **Keep pitch to 3 sentences** + deck link
4. **Reference their thesis** - Show you've done homework

### Tier 3: Content Strategy
1. **Publish technical content** on Mirror/Substack
2. **Podcast appearances** - Bankless, Unchained, The Block
3. **Research reports** - Partner with Messari, Delphi Digital
