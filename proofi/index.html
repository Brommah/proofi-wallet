For Developers
Integrate in
Minutes

Add decentralized credentials to any application with the Proofi SDK. Simple API, powerful capabilities.

app.js — Proofi SDK
// Install: npm install @proofi/sdk import { Proofi } from '@proofi/sdk'; // Initialize with your app ID const proofi = new Proofi({ appId: 'your-app-id', network: 'mainnet' }); // Authenticate a user const session = await proofi.auth.connect(); // Store a verifiable credential await proofi.credentials.store({ type: 'GameAchievement', data: { game: 'Space Defender', level: 42, score: 99500 }, signed: true }); // Verify any credential const result = await proofi.verify(credentialId); console.log(result.valid); // true console.log(result.issuer); // "5Grw..."

Email + PIN Authentication

No MetaMask, no seed phrases. Users sign in with email and a PIN. Keys are derived locally using PBKDF2 — the server never touches the private key.

Decentralized Storage

All data is stored on Cere's Decentralized Data Cloud (DDC). No single point of failure, no platform dependency, no vendor lock-in.
