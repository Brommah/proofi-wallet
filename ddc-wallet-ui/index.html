üì° API Reference

Available CereProof API endpoints. Base URL: /api

GET /api/status
Check API and DDC connection status, wallet info, and bucket configuration.
POST /api/store
Store raw data (text, JSON, or file) to Cere DDC. Returns a CID for permanent retrieval.
POST /api/credential/store
Store a client-signed verifiable credential on DDC. Accepts { credential, signature, issuer }.
POST /api/credential/issue
DEPRECATED ‚Äî Issue a server-signed credential. Use /credential/store with client-side signing instead.
POST /api/credential/verify
Verify a credential by CID. Checks signature validity, issuer identity, and on-chain storage.
POST /api/claim/submit
Submit a claim with supporting documents. Creates a pending request for an organization to review.
GET /api/claims/pending
List all pending claims awaiting organization review and approval.
GET /api/credentials/issued
List all credentials issued by the current wallet, with metadata and verification status.
GET /api/read/:cid
Read raw data from DDC by CID. Returns the stored content with metadata.
GET /api/profile/:wallet
Get public profile and credential history for a wallet address.
