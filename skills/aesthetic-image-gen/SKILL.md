# Aesthetic Image Generation Skill

Generate beautiful, trend-forward images using the latest 2026 design aesthetics.

## Quick Start

Build prompts using this structure:

```
[SUBJECT/CONCEPT] + [STYLE] + [COLOR PALETTE] + [LIGHTING] + [COMPOSITION] + [TECHNICAL SPECS]
```

## 2026 Trend Styles

### 1. Warm Foundational (Premium/Stripe-like)
Soft organic shapes, warm gradients, cream backgrounds. Anti-crypto, premium feel.

**Keywords**: `soft organic shapes`, `warm gradient`, `sand terracotta amber coral`, `cream background`, `flowing fabric`, `premium aesthetic`, `luxury brand`, `abstract minimalist`

### 2. Plushcore / Soft 3D
Tactile, toy-like, squeezable textures. Friendly and approachable.

**Keywords**: `plush texture`, `soft matte material`, `rounded forms`, `toy-like`, `tactile surface`, `plushy aesthetic`, `squeezable`, `soft shadows`

### 3. High-Gloss Hyper-Real
Chrome, liquid metal, luxury product photography aesthetic.

**Keywords**: `chrome surface`, `liquid metal`, `glossy highlights`, `studio lighting`, `product photography`, `reflective`, `high-end render`, `caustics`

### 4. Organic Abstract
Flowing forms inspired by nature—mycelium, cells, water.

**Keywords**: `organic network`, `mycelium pattern`, `cellular structure`, `flowing forms`, `natural geometry`, `watercolor blend`, `biomorphic shapes`

### 5. Lo-Fi / Retro 3D
Intentionally simple, nostalgic, low-poly charm.

**Keywords**: `low poly`, `simple geometry`, `retro 3D`, `PS1 aesthetic`, `blocky textures`, `limited palette`, `nostalgic`, `pixel-adjacent`

### 6. Diorama / Micro-worlds
Miniature scenes, tilt-shift feel, contained worlds.

**Keywords**: `miniature scene`, `diorama`, `tilt-shift`, `isometric`, `tiny world`, `detailed miniature`, `model-like`, `contained scene`

### 7. Soft-Glow Gradients
Ethereal, dreamy, soft light halos.

**Keywords**: `soft glow`, `ethereal lighting`, `gradient wash`, `halo effect`, `dreamy atmosphere`, `diffused light`, `gentle radiance`

## Generation Tips

1. **Be specific about what to exclude**: "No text, no circuits, no space imagery"
2. **Reference brands for style**: "aesthetic like Stripe" or "Apple marketing feel"
3. **Specify format**: "Square 1:1" or "16:9 widescreen"
4. **Layer adjectives**: "soft, flowing, organic" > just "organic"
5. **Use material language**: "matte ceramic", "brushed metal", "liquid glass"

## Key Lesson: Illustrative > Pure Abstract

**Images should DEPICT the concept, not just evoke a mood.**

❌ Too abstract: "Soft flowing organic shapes in warm gradients"
✅ Illustrative: "AI entity with visible glowing brain, data ribbons orbiting around it"

For each image, ask: "Does this SHOW what the tweet is about?"

- **Memory/persistence** → Show layers building up, visible brain, data accumulating
- **Protection/encryption** → Show hands blocked, shields, keys protected
- **Production/real-world** → Show actual use case (drones, airports, etc.)
- **Network/infrastructure** → Show connected nodes, nervous system with agents

## Anti-Patterns to Avoid

- ❌ Pure abstract that doesn't relate to content
- ❌ Generic "futuristic" or "sci-fi" without specifics
- ❌ "Beautiful" or "stunning" (subjective, not descriptive)
- ❌ Overloading with conflicting styles
- ❌ Forgetting to specify background
- ❌ Not excluding unwanted elements (text, people, etc.)
