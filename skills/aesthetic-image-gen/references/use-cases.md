## Model-Specific Tips

### Gemini (Google)
- Handles complex prompts well
- Good at following style references
- Add: `high-quality 3D render` for better results
- Specify: `Square format 1:1` explicitly

### DALL-E (OpenAI)
- Conversational prompts work best
- Can iterate: "make it warmer" / "more minimal"
- Strong at text avoidance
- Good semantic understanding

### Midjourney
- Shorter, keyword-focused prompts
- Use `--ar 1:1` for aspect ratio
- Add `--style raw` for less stylization
- `--v 6.1` for latest version

### Stable Diffusion / Flux
- Negative prompts important: `--no text, watermark, signature`
- More technical prompts work well
- Specify model/checkpoint for consistency
