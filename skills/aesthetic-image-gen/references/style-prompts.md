## 1. Warm Foundational / Premium Abstract

**Best for**: Brand campaigns, fintech, SaaS, luxury products

### Template A: Flowing Waves
```
Abstract minimalist image. Soft flowing organic shapes in warm gradient colors: 
sand, terracotta, amber, and soft coral. Cream/off-white background. 
The shapes suggest [CONNECTION/GROWTH/FLOW] like gentle waves or flowing fabric. 
Premium, sophisticated aesthetic like Stripe or a luxury brand. 
No technology imagery, no circuits, no space. Abstract and foundational. 
No text. Square format 1:1.
```

### Template B: Protected Core
```
Abstract minimalist image. A small warm glowing sphere (amber/gold) protected 
by soft curved shapes, while larger cool grey abstract forms [CONTRAST ELEMENT] 
in the background. Warm terracotta and sand tones for the protected element, 
cool greys for contrast. Cream background. Premium aesthetic like a luxury 
brand campaign. Soft, organic shapes. No technology, no circuits. 
Abstract and foundational. No text. Square format 1:1.
```

### Template C: Connected Network
```
Abstract minimalist image. Interconnected organic shapes suggesting a 
[NETWORK/SYSTEM/FLOW] in motion. Warm gradient colors: terracotta, amber, 
coral, sand. Flowing lines connecting circular forms. Represents something 
alive and working. Cream/off-white background. Premium, sophisticated aesthetic. 
Think Stripe or Apple marketing. No circuits, no tech imagery, no space. 
Pure abstract organic forms. No text. Square format 1:1.
```

### Template D: Organic Web
```
Abstract minimalist image. A vast interconnected web of soft organic shapes 
spreading across the canvas, like [NERVOUS SYSTEM/MYCELIUM/ROOTS] rendered 
in warm colors. Amber, terracotta, coral, and sand gradients. Cream background. 
The network feels alive, foundational, essential. Premium luxury aesthetic. 
No technology, no circuits, no digital elements. Pure organic abstract art. 
No text. Square format 1:1.
```

## Quick Modifiers

Add these to any prompt for specific effects:

| Effect | Add to prompt |
|--------|--------------|
| More depth | `layered composition with foreground and background elements` |
| More texture | `subtle grain texture, tactile surface quality` |
| More contrast | `dramatic lighting with deep shadows and bright highlights` |
| More movement | `dynamic flowing forms suggesting motion` |
| More calm | `still, balanced, symmetrical composition, meditative` |
| More premium | `luxury brand aesthetic, editorial quality, Vogue-worthy` |
| More playful | `whimsical, joyful, toy-like, approachable` |
